# integer_factorization_algorithms
There ara some implementations of algorithms used in integer factorization.
1. Simple factorization algorithm
2. Fermat's algorithm
3. Pollard's p-1 algorithm
4. Pollard's Rho algorithm
5. Quadratic Sieve algorithm

It is implemented with Javascript programming language.
Also is used HTML5, CSS, Bootstrap and Jquery.
